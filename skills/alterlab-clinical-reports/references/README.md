# Clinical Reports — Reference Index and External Standards

The full resource index (references, templates, scripts), workflows, and the
critical de-identification caveat live in the skill's top-level `SKILL.md`. This
file is a quick map of the `references/` directory plus authoritative external
links, so the rest of that material is not duplicated here.

## Reference files (`references/`)

- `care_report_sections.md` — CARE element-by-element, journal requirements, 18 HIPAA identifiers
- `case_report_guidelines.md` — CARE guidelines, de-identification, journal requirements, writing tips
- `diagnostic_report_templates.md` — radiology/pathology/lab section templates
- `diagnostic_reports_standards.md` — ACR, CAP, CLSI standards, structured reporting systems (BI-RADS, etc.)
- `clinical_trial_report_structures.md` — SAE/CSR/deviation component structures
- `clinical_trial_reporting.md` — ICH-E3, CONSORT, SAE reporting, MedDRA coding
- `patient_record_formats.md` — SOAP/H&P/discharge section formats
- `patient_documentation.md` — SOAP, H&P, discharge, coding
- `regulatory_compliance.md` — HIPAA, 21 CFR Part 11, ICH-GCP, FDA
- `medical_terminology.md` — SNOMED CT, LOINC, ICD-10-CM, CPT, abbreviations
- `data_presentation.md` — clinical tables, figures, Kaplan-Meier curves, CONSORT diagrams
- `peer_review_standards.md` — review criteria for clinical manuscripts
