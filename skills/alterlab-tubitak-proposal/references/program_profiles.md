# ARDEB Program Profiles — caps, windows, eligibility

**Every figure here is dated and changes each application period.** Before a user submits,
WebFetch the live program page and the current `başvuru rehberi` and reconcile. The skill's
structure checker restates this disclaimer on every run by design.

## 1001 — Bilimsel ve Teknolojik Araştırma Projelerini Destekleme Programı

(Support Programme for Scientific and Technological Research Projects)

| Item | Value | As of | Source |
|------|-------|-------|--------|
| Project duration | **≤ 36 months** ("Proje süresi en fazla 36 aydır") | 2026-1 period | program page (below) |
| Budget upper limit | **3,000,000 TRY** (proje destek üst limiti, **excl.** burs/scholarship, PTİ, kurum hissesi) | 2026-1 period | program page |
| Application window | Periodic **çağrı** (call) — open in announced periods | 2026-1 | program page |
| PI (Yürütücü) | Must hold a doctorate (or equivalent) and an eligible institutional affiliation | — | rehber |
| Submission | **PBS** `ardeb-pbs.tubitak.gov.tr`; **ARBİS** record must be current first | — | program page |

- Program page: `https://www.tubitak.gov.tr/tr/destekler/akademik/ulusal-destek-programlari/1001`
- Form: `https://tubitak.gov.tr/sites/default/files/2024-04/1001_basvuru_formu.doc`
- Guide: `https://tubitak.gov.tr/sites/default/files/2024-04/ardeb_1001_basvuru_rehberi.pdf`

## 1002-A — Hızlı Destek Modülü (Fast Support Module)

| Item | Value | As of | Source |
|------|-------|-------|--------|
| Project duration | **≤ 12 months** | current period | rehber |
| Budget upper limit | **150,000 TRY/yr** (**including** burs/scholarship) | 2026-02-01 | program page / announcement |
| Application window | **Rolling / year-round (sürekli)** | — | rehber |
| Form length | **≤ 12 pages** excluding annexes; Arial 9 | — | rehber |
| Scope | Short-term, low-budget, **non-urgent** R&D; excluded from the total project quota | — | rehber |

- Program page: `https://www.tubitak.gov.tr/tr/destekler/akademik/ulusal-destek-programlari/1002`
- Form: `https://tubitak.gov.tr/sites/default/files/20689/1002_a_basvuru_formu.doc`
  (newer dated form: `https://tubitak.gov.tr/sites/default/files/2025-04/1002_a_basvuru_formu_2025.doc`)
- Guide: `https://tubitak.gov.tr/sites/default/files/2024-04/1002_a_programi_basvuru_rehberi.pdf`

> TÜBİTAK has publicly announced **major changes to the 1002 program** ("1002-Hızlı Destek
> Programında Büyük Yenilik"). Treat 1002-A caps and structure as especially volatile and
> re-verify the current rehber before drafting.

## Related programs (out of scope here — for context only)

These are **not** scaffolded by this skill, but help route a misaimed request:

| Program | What it is | Note |
|---------|------------|------|
| **1002-B Acil Destek Modülü** | *Urgent* fast support | Different module from 1002-A; not scaffolded here |
| **3501 Kariyer (CAREER)** | Early-career development program | Eligibility: doctorate within ~7 years; separate form |
| **1005 / 1003 / 1004 / 1007** | Other ARDEB program lines | Different forms; out of scope |
| **BİDEB fellowships (e.g. 2219, 2214-A)** | Individual researcher fellowships, not project grants | Different directorate (BİDEB), different forms |

The 3501/1005/etc. figures are intentionally **omitted** here because they were not
re-verified for the current period in this skill's build; do not state their caps without a
fresh check.

## Verification recipe (run before any submission)

1. WebFetch the program page (table above) → confirm duration + budget cap for the **current**
   period.
2. WebFetch / open the current `başvuru rehberi` → confirm form length, font, and any section
   changes.
3. Confirm the **PBS** call is open (1001) or that rolling submission is active (1002-A).
4. Confirm the **ARBİS** record is current before starting PBS entry.
