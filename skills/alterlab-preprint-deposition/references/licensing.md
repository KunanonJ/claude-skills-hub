# Preprint Licensing

The license you pick at deposit time is a **standing commitment**: Creative
Commons and CC0 licenses are **irrevocable for the versions already posted**.
Choose deliberately, and reconcile with any funder open-access mandate and your
target journal's policy *before* posting.

## arXiv license set

arXiv presents these options at submission (verified on
`info.arxiv.org/help/license`):

- **arXiv.org perpetual, non-exclusive license 1.0** — you keep copyright and
  grant arXiv a distribution license. Least journal-friction.
- **CC BY 4.0** — Creative Commons Attribution.
- **CC BY-SA 4.0** — Attribution-ShareAlike.
- **CC BY-NC-SA 4.0** — Attribution-NonCommercial-ShareAlike.
- **CC BY-NC-ND 4.0** — Attribution-NonCommercial-NoDerivatives.
- **CC0 1.0** — public-domain dedication ("CC Zero").

## bioRxiv / medRxiv license set

bioRxiv and medRxiv let authors **retain copyright** and select a reuse option
(verified on biorxiv.org / medrxiv.org): **CC BY**, **CC BY-NC**, **CC BY-ND**,
**CC BY-NC-ND**, **CC0**, or **No reuse / All rights reserved**. The chosen terms
govern downstream reuse. Posting assigns a DOI regardless of the license choice.

## SSRN / OSF

- **SSRN:** author selects terms under SSRN's posting agreement; reuse is more
  restricted by default than the CC-first servers.
- **OSF Preprints:** No license / CC0 / CC BY / CC BY-NC-ND, depending on the
  provider.

## What each license commits you to

| License | Reuse allowed | Notes |
|---------|---------------|-------|
| arXiv non-exclusive 1.0 | distribution by arXiv only | You keep copyright; safest when a journal might object to CC reuse |
| CC BY 4.0 | any reuse incl. commercial, with attribution | Maximum openness; **irrevocable**; required by many funders |
| CC BY-SA 4.0 | as CC BY + derivatives must share-alike | Copyleft; can complicate later commercial publishing |
| CC BY-NC-SA / CC BY-NC-ND | non-commercial only | May **fail** a Plan S / cOAlition S mandate (which wants CC BY) |
| CC BY-ND | no derivatives | Limits translations/adaptations |
| CC0 | public domain, no attribution required | Broadest; **irrevocable** |
| No reuse (bioRxiv/medRxiv) | all rights reserved | Most restrictive reuse option offered |

## Funder mandates

- **cOAlition S / Plan S** funders generally require **CC BY** (sometimes CC0)
  on the deposited version; NC/ND variants typically do **not** comply. Confirm
  the specific funder's policy before recommending an NC/ND license.
- For **TÜBİTAK**-funded outputs and the Turkish açık bilim mandate, the
  deposition target and embargo rules differ → defer to `alterlab-aperta`.
- For writing the grant's **data management plan** (which often dictates the
  license), defer to `alterlab-open-science`.

## Journal friction

- A permissive preprint license (CC BY) **cannot be revoked** on versions
  already posted. Some closed-access journals dislike a CC-BY preprint; if you
  may publish in such a venue, the **arXiv non-exclusive license** (or "No
  reuse" on bioRxiv/medRxiv) is the lowest-friction choice.
- Always check the target journal's preprint policy first → `journal_policy.md`.
- Never recommend a single license without stating the trade-off it imposes.
